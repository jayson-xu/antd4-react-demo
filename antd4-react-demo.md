## 项目初始化

1. `npx create-react-app antd4-react-demo`

2. `yarn add antd react-router-dom`

3. 在根目录新建`.env`文件

   ```
   PORT=6789
   ```

4. 在根目录下创建`.prettierrc.json`文件，内容如下：

   ```json
   {
     "printWidth": 120,
     "singleQuote": true,
     "arrowParens": "avoid",
     "trailingComma": "es5"
   }
   ```

5.
